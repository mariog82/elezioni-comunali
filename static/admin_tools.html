Importazione prioritaria consiglieri

ATTENZIONE: questo import cancella e ricarica da zero liste e candidati già presenti.

Import rapido dei consiglieri con supporto Coalizione. Questo import aggiorna automaticamente:

  • anagrafica candidato/lista;
  • voti della lista collegata;
  • grafici e statistiche.

CSV consiglieri totali

Formato obbligatorio:
Numero Lista;Nome Lista;Coalizione;Numero Candidato;Nome Candidato
Il Nome Candidato non può essere numerico.

Statistiche aggiornate automaticamente

I grafici usano i dati salvati sul server. Se importi le preferenze totali dei consiglieri, la web app aggiorna anche i voti delle liste collegate sommando le preferenze importate.

Importazione voti totali

Carica i CSV dei risultati complessivi, senza colonna Sezione. Tutti i file devono usare il separatore ;.

Voti totali candidati sindaco

Formato: Numero Sindaco;Candidato Sindaco;Voti validi;Voti solo Sind

Voti totali liste

Formato: Numero Liste;Nome Lista;Voti validi

Preferenze totali consiglieri

Formato: Numero Liste;Nome Lista;Numero Candidato;Nome Candidato;Voti validi

Importazione schede nulle e bianche

Carica i CSV relativi a voti nulli, schede nulle, schede bianche e voti contestati/non assegnati. Tutti i file devono usare il separatore ;.

Schede totali

Formato: Voti nulli;Schede nulle;Schede bianche;V.cont.NoAss.

Schede per sezione

Formato: Sezione;Voti nulli;Schede nulle;Schede bianche;V.cont.NoAss.

Importazione voti sezione

Carica i CSV dei risultati per singola sezione/seggio. Tutti i file devono usare il separatore ;.

Voti candidati sindaco per sezione

Formato: Sezione;Numero Sindaco;Candidato Sindaco;Voti validi;Voti solo Sind

Voti liste per sezione

Formato: Sezione;Numero Liste;Nome Lista;Voti validi

Preferenze consiglieri per sezione

Formato: Sezione;Numero Liste;Nome Lista;Numero Candidato;Nome Candidato;Voti validi

Importa rappresentanti / utenti da CSV
